bass.winamp
===

[bass.winamp] is an extension enabling the use of Winamp input plugins with BASS.

Details
---
  - Installation command: ``Install-Package bass.winamp``

Content
---
  - Copies ``bass_winamp.dll`` during ``BeforeBuild``-targets to ``$(TargetDir)``
