TBS.CI.API
===

This NuGet package is for deploying the TBS TbsCIapi.dll to projects and GIT repositories.

Details
---
  - Installation command: ``Install-Package TBS.CI.API``

Content
---
  - Copies ``TbsCIapi.dll`` during ``BeforeBuild``-targets to ``$(TargetDir)``
